# EstimatePlus - Final project
## Webapp for creating cost estimates based on preloaded pricelists or self-added items. Allows to import excel (xlsx) pricelists and export estimates to excel.

## Tech used:
### - Spring Boot:
#### - Spring Web
#### - Spring Data JPA
#### - Spring Mail
### - Hibernate
#### - Forms validation
### - Java (8)
### - JSP
### - MySQL
### - JSTL
### - JavaScript
### - HTML + CSS (Bootstrap)
